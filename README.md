# Tic Tac Toe

## Platform: Android
## IDE: AndroidStudio
This app was created using Android Studio.
It's a basic TicTacToe game that allows the player to play vs another player or the CPU.
CPU functionality runs in a different thread.

### Functionality
- Enter Names menu allows for creating a 1 or 2 player game by entering the amount of names required.
- Play Game runs a basic game of TicTacToe where player 1 is X, player 2 is 0 (Player 2 could be a person or CPU)
- Standings keeps track of the scores of players via SharedPreferences and sorts them highest to lowest.
